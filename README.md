# 读书笔记共享管理系统 java239

本文档旨在介绍一个基于SpringBoot和Vue的前后端分离的读书笔记共享管理系统。

## 项目概述

本项目是一个功能全面的读书笔记共享平台，旨在为用户提供一个便捷的笔记分享与管理环境。系统主要分为用户和超级管理员两种角色，具备丰富的功能模块。

## 技术栈

- 前端：Vue
- 后端：SpringBoot
- 数据库：MySQL
- 开发工具：Idea，vscode
- 其他：mybatis-plus，element

## 功能模块

### 用户角色

- 个人笔记管理
- 笔记分享管理
- 交流管理
- 个人中心

### 超级管理员

- 公告管理
- 用户管理
- 个人后台

## 系统角色

- 用户：可以管理个人笔记，分享笔记，参与交流。
- 超级管理员：负责管理所有用户和公告，维护系统运行。

## 运行环境

- 前端：现代浏览器
- 后端：Java 1.8+
- 数据库：MySQL 5.7+
